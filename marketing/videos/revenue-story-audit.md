**Narration (verbatim):**

"The skill file is one markdown document. Top of the file is the frontmatter — name, version, when to use it, when not to use it. The 'when not to use' block is the part I want you to read. That block exists because the most expensive mistake operators make with this framework is running it without the people who control the answers. The skill says no for you, so you do not have to. Below that, nine facilitation segments. The dollar trace. Board deck lie detector. Concentration heat map. Growth assumption autopsy. Uncomfortable metric anonymous reveal. Two-version reconciliation. Each segment is one move, run in order, producing one artifact. The order matters. The audit does not work if you skip ahead."

**Camera:** screen-focused. Right half (Claude Code) becomes the focus.
**Visual cues:** open `examples/revenue-story-audit/input.md`. Copy the full contents. Paste into Claude Code with the revenue-story-audit skill loaded. Hit enter. Let the response stream live. Point to the screen as each segment appears.

**Narration (verbatim):**

"This is the worked example. NorthStack. Fintech operations layer for mid-market neobanks. Twenty million ARR. Series B, prepping Series C in ninety days. I am pasting their full intake into Claude Code right now. The skill is loaded. Watch what happens. Claude does not jump to a reconciliation. The first thing it does is confirm the data inventory — four quarters of revenue by customer, channel, product, geography, cohort. Two board decks. Last sales pitch. Top 5 expansion deals. Top 5 churn losses with reasons. NorthStack has all of it. Claude proceeds. Now it runs the $1 Trace. Pick a single dollar from last quarter. Trace it from origin to recognition. Document every assumption. The first dollar Claude picks is from a renewal — assumed-outbound on the channel sheet, actually a partner-sourced renewal where the partner stopped returning emails three months ago. That single dollar surfaces a dead referral channel nobody had named yet. Then the Board Deck Lie Detector. Line by line. The 'plus one hundred twenty percent Y/Y' claim — extrapolated from one strong quarter, not trailing. Reclassified as hopeful. NRR claimed bullish at one hundred fifteen — actual cohort math says ninety-six. Reclassified as wrong. This is what the skill does. It produces a sequence of findings, each one defensible, each one referenced to evidence in the intake."
