# Code of Conduct

Adversarial in frameworks. Professional with humans.

## Standards

- Critique frameworks, not contributors
- Cite evidence, not opinions, in PR reviews
- Assume good faith on first contact; revoke after demonstrated bad faith
- No harassment, no ad hominem, no bad-faith reviews

## Enforcement

Issues reported to phil@petrichorgrowth.com. Maintainer judgment is final. Outcomes: warning, PR closure, repository ban.

Adapted from the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
