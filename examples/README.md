# Worked Examples — Resilience Stack

Each skill has an anonymized worked example: input context, full skill run trajectory, all 5 deliverables. Use these to:

- Understand what a "good" output looks like before running the skill
- Train new facilitators
- Calibrate expectations with clients

## Format

```
examples/
├── <skill-name>/
│   ├── input.md                  # company context provided to the skill
│   ├── trajectory.md             # phase-by-phase facilitation log
│   ├── deliverable-1.md          # produced deliverable
│   ├── deliverable-2.md
│   ├── deliverable-3.md
│   ├── deliverable-4.md
│   ├── deliverable-5.md          # scorecard
│   └── post-engagement-notes.md  # what worked, what surprised, follow-up
└── README.md
```

## Anonymization

All company names → "Synthetic Co" or similar. Revenue figures rounded to nearest $5M. Customer names → "Customer A/B/C". Industry kept ("vertical SaaS for legal", "DTC pet wellness") but specific competitors masked.

## Contribution

Worked examples are the highest-value contribution after new skills. Open a PR with the full structure above. Petrichor reviews within 14 days.
